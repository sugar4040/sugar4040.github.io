# Product Overview

回遊 (Kaiyuu) is an HTML-based mystery-solving event system designed for interactive puzzle events. The system is configuration-driven, allowing event organizers to define puzzles and content through external JSON files without modifying code.

## Key Features

- Configuration-driven puzzle system with JSON-based event definitions
- Offline-capable with client-side operation and local storage persistence
- Japanese language support (UTF-8 encoding with Japanese fonts)
- Accessible design following WCAG guidelines with semantic HTML5 and ARIA labels
- Responsive mobile-first design supporting 320px to desktop widths
- Zero external dependencies (vanilla JavaScript, HTML5, CSS3)

## Target Users

Event organizers who need a simple, self-contained system for running mystery-solving or puzzle-based events that can work offline and on mobile devices.
